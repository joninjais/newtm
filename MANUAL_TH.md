## ข้อมูลทั่วไป

**NewTM** คือระบบจัดการข่าวแบบตารางที่ออกแบบมาสำหรับหน่วยงานราชการ โปรแกรมนี้สามารถ:
- ✓ สร้างและจัดการข่าวจากผู้ดูแลระบบ
- ✓ แบ่งข่าวตามหมวดหมู่ (Categories)
- ✓ แสดงข่าวในหลายรูปแบบ (Grid, Table, List)
- ✓ รองรับ Elementor Editor
- ✓ มี Shortcodes สำหรับการแสดงข่าวแบบกำหนดเอง
- ✓ ระบบแคชอัจฉริยะ (Smart Caching)

---

## การติดตั้งและเปิดใช้งาน

### 1. ติดตั้ง Plugin
```
1. เข้า WordPress Dashboard
2. ไปที่ "Plugins" → "Add New"
3. ค้นหา "NewTM"
4. คลิก "Install Now" และ "Activate"
```

### 2. ตรวจสอบสถานะการติดตั้ง
```
1. ไปที่ WordPress Admin
2. ค้นหา "ข่าวสาร" ในเมนูด้านซ้าย
3. ถ้าเห็นเมนู "ข่าวสาร" แสดงว่าติดตั้งสำเร็จ
```

## การสร้างข่าว

### วิธีสร้างข่าวใหม่

**ขั้นตอนที่ 1: เข้าหน้าสร้างข่าว**
```
1. ไปที่ WordPress Admin
2. คลิก "ข่าวสาร" → "เพิ่มข่าวใหม่"
```

**ขั้นตอนที่ 2: กรอกข้อมูลข่าว**
```
📝 หัวข้อ (Title)
   - ป้อนหัวข้อข่าวที่ชัดเจนและเข้าใจง่าย
   - ตัวอย่าง: "ประกาศรับสมัครสอบเพื่อบรรจุเข้าราชการ ประจำปี 2567"

📄 เนื้อหา (Content)
   - เขียนเนื้อหาข่าวให้ครบถ้วน
   - สามารถเพิ่มรูปภาพ, ตาราง, ลิงก์ได้
   - ใช้ Gutenberg Editor หรือ Elementor ได้ทั้งสองอย่าง

🖼️ รูปภาพเด่น (Featured Image)
   - ให้ตั้งรูปภาพเด่นสำหรับข่าวนี้
   - ขนาดแนะนำ: 1200x600 พิกเซล
   - จะแสดงในรูป Grid หรือ Card

📚 หมวดหมู่ (Categories)
   - เลือกหมวดหมู่ของข่าว (ต้องเลือกอย่างน้อย 1 รายการ)
   - ถ้าไม่มีหมวดหมู่ให้เพิ่มใหม่

🏷️ แท็ก (Tags)
   - (ทางเลือก) เพิ่มแท็กเพื่อจัดหมวดหมู่เพิ่มเติม

💬 สรุปย่อ (Excerpt)
   - (ทางเลือก) ป้อนสรุปย่อของข่าว (จะแสดงในรูป Grid)
```

**ขั้นตอนที่ 3: เผยแพร่ข่าว**
```
1. ตรวจสอบให้แน่นอนว่ากำหนดหมวดหมู่แล้ว
2. คลิก "Publish" (เพื่อเผยแพร่)
   หรือ "Save as Draft" (เพื่อบันทึกเป็นแบบร่าง)
```

### ⚠️ สิ่งสำคัญที่ควรทำ
- ✓ **ต้องตั้งหมวดหมู่** - ข่าวจะไม่แสดงถ้าไม่มีการกำหนดหมวดหมู่
- ✓ **ต้องเผยแพร่** - ข่าว Draft จะไม่แสดงให้ผู้อื่นเห็น
- ✓ **ตั้งรูปเด่น** - จะทำให้ข่าวดูน่าดึงดูด

## การจัดการหมวดหมู่

### สร้างหมวดหมู่ใหม่
```
1. ไปที่ WordPress Admin
2. คลิก "ข่าวสาร" → "หมวดหมู่"
3. คลิก "เพิ่มหมวดหมู่ใหม่"
4. กรอก:
   - ชื่อหมวดหมู่: เช่น "ข่าวสำคัญ", "ประกาศรับสมัคร" เป็นต้น
   - Slug: ไม่ป้อนก็ได้ (ระบบจะสร้างอัตโนมัติ)
   - คำอธิบาย: (ทางเลือก) เพิ่มคำอธิบายหมวดหมู่
5. คลิก "บันทึก"
```

### ตั้งค่า Items Per Page สำหรับแต่ละหมวดหมู่
```
1. ไปที่ WordPress Admin
2. คลิก "ข่าวสาร" → "หมวดหมู่"
3. ลากเมาส์ไปที่หมวดหมู่ที่ต้องการ → คลิก "แก้ไข"
4. ในส่วน "จำนวนข่าวต่อหน้า" ให้ป้อนจำนวนที่ต้องการ
5. คลิก "บันทึก"
```

## การใช้งาน Widgets

### Widget 1: ข่าว Grid (News Grid)
**คำอธิบาย**: แสดงข่าวในรูป Grid/Card ที่ดึงดูดสายตา

**การใช้งาน**:
```
1. เข้า Elementor Editor (หรือใช้ Gutenberg)
2. ค้นหา Widget ชื่อ "ข่าว Grid"
3. ลาก Widget เข้าไปในหน้า
4. ตั้งค่า:
   - จำนวนข่าวที่แสดง: 6 (สามารถเปลี่ยนได้)
   - จำนวนคอลัมน์: 3 (1, 2, 3, 4, หรือ 6)
   - เรียงลำดับตาม: วันที่ (เก่า→ใหม่) / ชื่อเรื่อง / สุ่ม
   - ลำดับ: ใหม่ → เก่า (แนะนำ)
   - กรองตามหมวดหมู่: เลือกหมวดหมู่เฉพาะ หรือทั้งหมด
   - แสดง: รูปภาพ, เนื้อหาย่อ, วันที่, หมวดหมู่
```

**ตัวอย่างการใช้งาน**:
- ส่วนเว็บแรก (Hero Section) → จำนวนข่าว 3-4 อัน, 1-2 คอลัมน์
- บริเวณตรงกลาง → จำนวนข่าว 6-9 อัน, 3 คอลัมน์
- Sidebar → จำนวนข่าว 5 อัน, 1 คอลัมน์

---

### Widget 2: ข่าว Table (News Table)
**คำอธิบาย**: แสดงข่าวในรูป Table/List ที่เป็นระเบียบ

**การใช้งาน**:
```
1. เข้า Elementor Editor
2. ค้นหา Widget ชื่อ "ข่าว Table"
3. ลาก Widget เข้าไปในหน้า
4. ตั้งค่า:
   - จำนวนข่าวที่แสดง: 10
   - เรียงลำดับตาม: วันที่
   - ลำดับ: ใหม่ → เก่า
   - กรองตามหมวดหมู่: (เลือก)
   - แสดง: รูปภาพ, หมวดหมู่, ป้าย NEW
   - ป้าย NEW แสดงกี่วัน: 7 วัน
```

**เหมาะสำหรับ**:
- ตารางข่าวหลัก
- รายการข่าวแบบสั้น ๆ
- หน้า Archive

---

### Widget 3: Category Tabs (หมวดหมู่แบบ Tabs)
**คำอธิบาย**: แสดงข่าวแบบ Tabs ตามหมวดหมู่

**การใช้งาน**:
```
1. เข้า Elementor Editor
2. ค้นหา Widget ชื่อ "Category Tabs"
3. ลาก Widget เข้าไปในหน้า
4. Widget จะแสดง Tabs สำหรับแต่ละหมวดหมู่
   - คลิก Tab → แสดงข่าวของหมวดหมู่นั้น
```

## การใช้งาน Shortcodes

### Shortcode 1: แสดงข่าวตามหมวดหมู่
```
[newtm_table category="slug-ของ-หมวดหมู่" limit="5" show_title="true"]
```

**คำอธิบาย**:
```
📌 category: (บังคับ)
   - Slug ของหมวดหมู่
   - ตัวอย่าง: news, announcement, news-important
   - ดูได้จาก: ข่าวสาร → หมวดหมู่ → (มองหาคอลัมน์ "Slug")

📌 limit: (ทางเลือก)
   - จำนวนข่าวที่จะแสดง
   - ค่าเริ่มต้น: 5 อัน
   - ตัวอย่าง: limit="10"

📌 show_title: (ทางเลือก)
   - แสดงชื่อหมวดหมู่หรือไม่
   - ค่า: "true" (แสดง) หรือ "false" (ไม่แสดง)
   - ค่าเริ่มต้น: "true"

📌 title_length: (ทางเลือก)
   - จำนวนตัวอักษรสูงสุดของหัวข้อ
   - ค่าเริ่มต้น: ตามตั้งค่า
   - ตัวอย่าง: title_length="50"
```

**ตัวอย่างการใช้**:
```html
<!-- แสดงข่าวหมวดหมู่ "ข่าวสำคัญ" 5 อัน -->
[newtm_table category="news-important" limit="5"]

<!-- แสดงข่าวหมวดหมู่ "ประกาศ" 10 อัน ไม่แสดงชื่อหมวดหมู่ -->
[newtm_table category="announcements" limit="10" show_title="false"]

<!-- แสดงข่าว และจำกัดหัวข้อที่ 60 ตัวอักษร -->
[newtm_table category="news" limit="8" title_length="60"]
```

---

### Shortcode 2: แสดงข่าวทั้งหมด (ทั้งหมดของหมวดหมู่)
```
[newtm_all_categories columns="4" limit="0"]
```

**คำอธิบาย**:
```
📌 columns: (ทางเลือก)
   - จำนวนคอลัมน์
   - ค่าเริ่มต้น: 4

📌 limit: (ทางเลือก)
   - จำนวนข่าวต่อหมวดหมู่
   - ค่า 0 = ไม่จำกัด
   - ค่าเริ่มต้น: 0
```

**ตัวอย่างการใช้**:
```html
<!-- แสดงทั้งหมดแบบ 4 คอลัมน์ -->
[newtm_all_categories columns="4"]

<!-- แสดง 3 ข่าวต่อหมวดหมู่ -->
[newtm_all_categories limit="3"]

<!-- แสดง 2 คอลัมน์, 5 ข่าวต่อหมวด -->
[newtm_all_categories columns="2" limit="5"]
```

## การตั้งค่า

### เข้าหน้าตั้งค่า
```
1. ไปที่ WordPress Admin
2. คลิก "ข่าวสาร" → "ตั้งค่า"
```

### ตัวเลือกการตั้งค่า

| ตัวเลือก | คำอธิบาย | ค่าเริ่มต้น |
|---------|---------|-----------|
| **จำนวนตัวอักษรสูงสุดของหัวข้อ** | จำกัดความยาวของหัวข้อในรายการ (0 = ไม่จำกัด) | 0 |
| **จำนวนตัวอักษรสูงสุดของเนื้อหาย่อ** | จำกัดความยาวของ Excerpt (0 = ไม่จำกัด) | 0 |
| **จำนวนข่าวต่อหน้า** | จำนวนข่าวที่แสดงต่อหน้า (ค่าเริ่มต้นสำหรับ Shortcodes) | 10 |

## การ Troubleshooting

### ❌ ปัญหา: ข่าวไม่แสดง

**สาเหตุที่เป็นไปได้**:
1. ข่าวยังเป็น Draft (แบบร่าง)
2. ข่าวไม่มีการกำหนดหมวดหมู่
3. ข่าวเก่าถูกลบ
4. มีปัญหากับ Cache

**วิธีแก้**:
```
✓ ตรวจสอบสถานะ Post
  1. ไปที่ WordPress Admin
  2. คลิก "ข่าวสาร" → "ข่าวทั้งหมด"
  3. ตรวจสอบสถานะว่าเป็น "Publish" หรือ "Draft"
  4. ถ้า Draft ให้คลิก "Edit" แล้วคลิก "Publish"

✓ ตรวจสอบหมวดหมู่
  1. ไปที่ WordPress Admin
  2. คลิก "ข่าวสาร" → "ข่าวทั้งหมด"
  3. เลื่อนไปดูคอลัมน์ "หมวดหมู่"
  4. ถ้าว่าง ให้คลิก "Edit" แล้วเพิ่มหมวดหมู่

✓ Clear Cache
  1. ไปที่ WordPress Admin
  2. ใช้ Plugin Clear Cache (ถ้ามี)
  3. หรือกด Ctrl+F5 บนเบราว์เซอร์
```

---

### ❌ ปัญหา: Widget ไม่แสดงตัวเลือกการตั้งค่า

**สาเหตุ**:
- Elementor ยังไม่โหลด widgets ทั้งหมด

**วิธีแก้**:
```
1. รีเฟรชหน้า (Ctrl+F5)
2. ปิด Elementor แล้วเปิดใหม่
3. ทำการ Deactivate แล้ว Activate Plugin ใหม่
```

---

### ❌ ปัญหา: ข่าวแสดงซ้ำ

**สาเหตุ**:
- บางทีอาจเป็น Plugin ที่ขัดแย้ง
- ปัญหากับ Query

**วิธีแก้**:
```
1. ลองปิดเปิด Widget อีกครั้ง
2. ทำการ Clear Browser Cache
3. Check ว่า Posts ไม่ได้มี Duplicate Status
```

---

### ❌ ปัญหา: Shortcode ไม่ทำงาน

**สาเหตุ**:
- Slug หมวดหมู่ผิด
- Plugin ไม่ได้เปิดใช้งาน

**วิธีแก้**:
```
1. ตรวจสอบ Slug ของหมวดหมู่
   - ไปที่ ข่าวสาร → หมวดหมู่
   - ลากเมาส์ไปที่หมวดหมู่ → ดูขึ้นของ URL
   - Slug อยู่ที่ส่วน category=<slug>

2. ก็อปปี้ Slug ที่ถูกต้องไปใส่ใน Shortcode
   [newtm_table category="ใส่-slug-ที่ถูกต้อง"]
```

---

### 🔧 ตรวจสอบสถานะข่าว (สำหรับ Admin)

**วิธีดู Widget Status**:
```
1. ไปที่ ข่าวสาร → ตั้งค่า
2. ดูส่วน "NewTM Widget Status"
3. ตรวจสอบ:
   - Published News: ควรมี > 0
   - Unassigned Categories: ควรเป็น 0
   - Problematic Posts: ควรเป็น 0
```

---

### 🔍 ค้นหาข้อมูล Slug ของหมวดหมู่

```
1. ไปที่ WordPress Admin
2. คลิก "ข่าวสาร" → "หมวดหมู่"
3. ลากเมาส์ไปที่หมวดหมู่ที่ต้องการ
4. ดูบ้านล่างซ้าย (URL) จะเห็น: category&tag=newtm_category&post_type=newtm_news&s=&name=<SLUG>
   SLUG ก็คือสิ่งที่อยู่หลัง name=
```
